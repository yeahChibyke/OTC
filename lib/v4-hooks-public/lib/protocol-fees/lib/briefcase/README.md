## Uniswap Briefcase

[![NPM Version](https://img.shields.io/npm/v/@uniswap/briefcase.svg?style=flat)](https://www.npmjs.com/package/@uniswap/briefcase)

Uniswap briefcase is a collection of everything you need to build on Uniswap. It includes interfaces and libraries for all Uniswap protocol smart contracts, as well as deployer libraries to quickly deploy Uniswap protocol smart contracts within your unit tests.
